# submissions 안내

이 폴더는 스터디 풀이 제출이 모이는 공간입니다.  
**폴더 구조는 강제**이며, 구조가 깨지면 리뷰 자동 요청(CODEOWNERS)이 정상 동작하지 않습니다.

---

## 제출 경로(강제)

- GroupA: `submissions/groupA/{GitHubID}/Wxx/`
- GroupB: `submissions/groupB/{GitHubID}/Wxx/`

---

## 권장 파일 구성

Wxx 폴더 안에 문제 풀이 파일을 5개 넣습니다.

예시:
- `boj_XXXXX.java`
- `pgs_YYYYY.py`
- `lc_zzzzz.cpp`

선택:
- 각 문제별 간단한 설명을 남기고 싶다면
  `docs/templates/problem-readme.md` 템플릿을 복사해 사용하세요.
