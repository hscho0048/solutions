## 9-6. `docs/review-checklist.md`

```md
# 리뷰 체크리스트

PR을 승인하기 전에 아래 항목을 확인합니다.

## 1) 기본
- [ ] 제출 경로가 규칙에 맞는가?
  - GroupA: submissions/groupA/{GitHubID}/Wxx/
  - GroupB: submissions/groupB/{GitHubID}/Wxx/
- [ ] 주당 5문제 제출이 맞는가?
- [ ] 파일명/확장자가 정상인가?

## 2) 풀이 품질
- [ ] 시간 복잡도 설명이 가능할 정도로 타당한가?
- [ ] 불필요한 중복 로직이 없는가?
- [ ] 예외/엣지 케이스를 고려했는가?
- [ ] 변수명/함수명이 의미를 담고 있는가?

## 3) 커뮤니케이션
- [ ] 리뷰 코멘트에 성실히 응답했는가?
- [ ] 수정이 필요한 부분을 반영했는가?

## 승인 기준
- “완벽”이 아니라 “이대로 학습 저장소에 남겨도 되는 수준”이면 승인합니다.
- 단, 경로 규칙 위반은 승인하지 않습니다.
