# 제출 방법 (Fork → PR)

## 0) 전제
- 본인은 Organization 멤버로 초대되어 있어야 합니다.
- 본인 그룹(GroupA / GroupB)에 맞는 폴더로 제출해야 합니다.

---

## 1) Fork 하기
1. `ORG_NAME/solutions` 레포 접속
2. 우측 상단 **Fork** 클릭
3. 본인 계정으로 Fork 생성

---

## 2) 로컬로 클론
```bash
git clone https://github.com/<YOUR_GITHUB_ID>/solutions.git
cd solutions
